Terminal music player for YouTube

One binary. Full terminal playback.

Termphonic is a focused TUI music player for Linux. It searches YouTube, manages a queue, restores playback state, and ships as a single executable with embedded runtimes.

Download latest release View source
Release model Single file
Runtime Embedded yt-dlp + Deno
Platform Linux

What it does

Built for keyboard-first playback

Search and queue

Search YouTube, browse paginated results, and queue tracks without leaving the terminal.

Playback controls

Play, pause, stop, seek, change volume, and switch between off, shuffle, and single repeat modes.

Session restore

Resume active playback after reopening the app, including queue, position, and volume.

Responsive TUI

Compact layouts, readable duration labels, queue highlighting, and a CAVA-style visualizer.

Single-file release

No Python, no Node, no install-time runtime setup. The binary extracts what it needs on first launch.

Portable source install

From source, the installer still works locally and installs to `~/.local` with the same app name.

Install

Simple install path

curl -L -o termphonic \
  https://github.com/bocd3r/Termphonic/releases/latest/download/termphonic
chmod +x termphonic
./termphonic

The release binary carries `yt-dlp` and Deno inside itself and extracts them into the user cache automatically. 

$ termphonic
Search / queue / play
Shuffle, session restore, and single-file release
